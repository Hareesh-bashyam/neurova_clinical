# Post-Market Surveillance (PMS) Plan

## Objective
To monitor the safety, performance, and reliability of Neurova Clinical Engine
after deployment.

## Metrics Monitored
- Assessment completion rate
- Session abandonment rate
- Frequency of red-flag triggers
- System errors during scoring or report generation
- User-reported complaints

## Monitoring Frequency
- Monthly internal review
- Immediate review for any reported adverse event

## Adverse Event Definition
An adverse event includes:
- Incorrect score generation due to system error
- Failure to display red-flag warnings
- Unauthorized access to reports
- Report delivery failure in clinical workflows

## Corrective Actions
- Root cause analysis
- Documentation of corrective action
- Software patch or configuration update if required
- Notification to relevant stakeholders when applicable

## Record Retention
All PMS records are retained for a minimum of 5 years.
