# Red Flag Handling Protocol

## Definition
A red flag indicates potential safety risk requiring immediate clinician attention.

## Examples
- Self-harm ideation indicators
- Extremely high symptom burden scores

## System Behavior
- Red flag highlighted prominently in report
- Report marked as "Clinician Review Required"
- Patient instructed to return to clinician/staff

## Mandatory Facility Requirements
1. Immediate escalation to licensed clinician
2. Documentation of clinician acknowledgment
3. No standalone use without mental health support

## Contraindication
This system is not intended for emergency or crisis intervention.
